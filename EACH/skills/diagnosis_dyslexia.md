# Dyslexia — Teaching Guide for EACH Agents

> ⚠️ EACH never diagnoses. This student's Dyslexia diagnosis was set by the admin.
> Your role is to help the teacher support this student — not to assess or confirm the diagnosis.

## Understanding Dyslexia in the Classroom

Dyslexia is a language-based learning difference that primarily affects reading and writing.
It is NOT related to intelligence — dyslexic students often have strong verbal reasoning,
creativity, and problem-solving skills.

Key challenges:
- **Decoding** — difficulty sounding out unfamiliar words
- **Reading fluency** — slow, effortful reading even with decoding ability
- **Spelling** — inconsistent, phonetically unusual errors
- **Working memory** — difficulty holding verbal sequences (instructions, phone numbers)
- **Written expression** — ideas are rich verbally but hard to get on paper

Arabic dyslexia has specific characteristics:
- Root-pattern morphology makes word recognition harder
- Short vowels (harakat) are often omitted in text, increasing difficulty
- Right-to-left directionality adds complexity when switching between AR/EN

## Assessment Agent — Key Questions to Ask

**Reading**
- "What reading level is [student] working at compared to peers?"
- "Does [student] struggle more with decoding individual words or reading fluency?"
- "How is [student]'s reading in Arabic vs English — is one stronger?"

**Writing**
- "Does [student] avoid writing tasks or rush through them?"
- "What do their spelling errors look like — random or phonetically close?"
- "Can [student] express ideas verbally that they struggle to write down?"

**Strategies Already Tried**
- "Have you used audiobooks or text-to-speech? Did it help engagement?"
- "Has color overlay, larger font, or dyslexia-friendly font been tried?"
- "Does [student] use any assistive technology?"

**Support Needed**
- "Is the priority reading, writing, or both this week?"
- "Are there specific subjects where reading/writing demands are highest?"

## Planning Agent — Evidence-Based Strategies

### High-Impact Strategies
| Strategy | Best For | How to Use |
|---|---|---|
| **Multisensory instruction** | Decoding | See it + say it + write it + trace it |
| **Audiobooks / text-to-speech** | Reading access | Provide alongside print, not instead of |
| **Dyslexia-friendly formatting** | All reading tasks | Larger font, increased spacing, OpenDyslexic font |
| **Oral response option** | Written tasks | Allow verbal answers to demonstrate knowledge |
| **Graphic organizers** | Writing planning | Visual structure before writing begins |
| **Extended time** | All timed tasks | 1.5× standard time minimum |
| **Phonological awareness games** | Decoding foundation | Rhyming, blending, segmenting sounds |
| **Word banks** | Spelling support | Provide key vocabulary on desk |

### Arabic-Specific Strategies
- Provide texts with full harakat (short vowels) to reduce decoding load
- Use root-pattern practice to build morphological awareness
- Allow switching to English for expression when Arabic writing is blocking

### Weekly Plan Structure for Dyslexia
- **Day 1**: Introduce new vocabulary with multisensory method (visual + auditory + kinesthetic)
- **Day 2**: Reading with support (paired reading, audiobook + text)
- **Day 3**: Writing with scaffold (graphic organizer → draft → teacher support)
- **Day 4**: Fluency practice (re-reading familiar text, timed short passages)
- **Day 5**: Oral or project-based assessment (reduces writing barrier)

### What to AVOID in Dyslexia Plans
- Cold-calling student to read aloud unexpectedly
- Penalizing spelling errors in content-area subjects
- Copying from board (high working memory load)
- Timed tests without accommodation
- Comparing reading speed to peers

## Reflection Agent — Quality Checklist for Dyslexia Plans

Before approving a plan, verify:
- [ ] At least one multisensory learning activity per week
- [ ] Reading accommodation mentioned (audiobook, paired reading, or supported reading)
- [ ] Writing tasks have a scaffold or oral alternative
- [ ] No cold-read-aloud without student choice
- [ ] Arabic-specific considerations addressed if primary language is Arabic
- [ ] Extended time noted for any assessment activities
- [ ] Both Arabic and English versions are consistent

---

## Evaluation Agent — Scoring Notes

When scoring plans for Dyslexia students:
- **Score 5**: Plan removes reading/writing barriers while still building skills, multisensory approach throughout, oral alternatives offered
- **Score 4**: Plan reduces barriers in most tasks, some multisensory elements
- **Score 3**: Plan acknowledges dyslexia but limited practical accommodations
- **Score 1-2**: Plan makes no adjustments for reading/writing difficulties

Alert threshold: Agent score < 3 for two consecutive weeks → trigger `no_improvement` alert.
