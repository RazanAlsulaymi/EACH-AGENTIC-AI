# Processing Disorder — Teaching Guide for EACH Agents

> ⚠️ EACH never diagnoses. This student's Processing Disorder diagnosis was set by the admin.
> Your role is to help the teacher support this student — not to assess or confirm the diagnosis.

## Understanding Processing Disorders in the Classroom

"Processing Disorder" is an umbrella term. Common types include:

- **Auditory Processing Disorder (APD)** — difficulty understanding spoken language despite normal hearing
- **Visual Processing Disorder (VPD)** — difficulty interpreting visual information despite normal vision
- **Language Processing Disorder (LPD)** — difficulty attaching meaning to spoken words
- **Sensory Processing Disorder (SPD)** — difficulty regulating responses to sensory input

Ask the teacher which type applies (or refer to student's IEP/assessment file if uploaded).
Plans should be tailored to the specific processing challenge.

---

## Assessment Agent — Key Questions to Ask

**Understanding the Processing Challenge**
- "Does [student] struggle more with listening/spoken instructions or with reading/visual tasks?"
- "Do they need you to repeat instructions often? Do they ask for clarification?"
- "Is there a specific type of input that causes the most confusion?"

**Speed of Processing**
- "How long does [student] typically take to respond after a question?"
- "Do they understand eventually with more time, or do they remain confused?"
- "Are they overwhelmed in fast-paced activities?"

**Environment**
- "Does background noise significantly affect [student]'s ability to focus or understand?"
- "Is there a seating arrangement that helps them process better?"

**Strategies Already Tried**
- "Have you tried giving written instructions alongside verbal ones?"
- "Has extra processing time made a difference?"
- "Do visual aids or diagrams help more than verbal explanation?"

## Planning Agent — Evidence-Based Strategies

### High-Impact Strategies
| Strategy | Best For | How to Use |
|---|---|---|
| **Multi-modal instruction** | All types | Present information in 2+ formats simultaneously |
| **Wait time** | All types | Pause 10-15 seconds after asking a question |
| **Written + verbal instructions** | APD/LPD | Always pair spoken instructions with written version |
| **Pre-teaching vocabulary** | LPD | Introduce key terms before the lesson |
| **Visual organizers** | VPD/APD | Charts, diagrams, mind maps to support comprehension |
| **Reduced background noise** | APD/SPD | Preferential seating away from noise sources |
| **Chunked instruction** | All types | One step at a time, confirm understanding before continuing |
| **Repetition without rushing** | All types | Re-explain naturally, same words, slower pace |

### Type-Specific Additions

**For Auditory Processing (APD):**
- Seat student close to teacher, away from HVAC/windows
- Use FM system or sound-field system if available
- Provide written summary of verbal lessons

**For Visual Processing (VPD):**
- Reduce visual clutter on worksheets
- Use clean fonts, high contrast, adequate spacing
- Allow student to cover portions of text while reading

**For Sensory Processing (SPD):**
- Identify sensory triggers from teacher/parent report
- Provide sensory breaks before overstimulation occurs
- Offer sensory tools (noise-canceling headphones, fidgets)

### Weekly Plan Structure for Processing Disorders
- **Day 1**: Pre-teach key vocabulary/concepts (reduces processing load for new content)
- **Day 2**: Multi-modal instruction (visual + verbal + hands-on simultaneously)
- **Day 3**: Structured practice with reduced noise/distraction environment
- **Day 4**: Review with student-led explanation (checks real understanding)
- **Day 5**: Low-pressure assessment with extended time

### What to AVOID in Processing Disorder Plans
- Rapid-fire oral questioning
- Complex multi-step verbal instructions without written backup
- Assuming the student understood because they didn't ask questions
- Fast transitions without warning
- Overloading with visual information on a single page

## Reflection Agent — Quality Checklist for Processing Disorder Plans

Before approving a plan, verify:
- [ ] Instructions are always presented in at least two modalities
- [ ] Extended processing/wait time is built into daily activities
- [ ] Plan addresses the specific type of processing challenge (auditory/visual/sensory)
- [ ] Environmental considerations mentioned (seating, noise, visual clutter)
- [ ] No activities rely solely on rapid verbal response
- [ ] Both Arabic and English versions are consistent

---

## Evaluation Agent — Scoring Notes

When scoring plans for Processing Disorder students:
- **Score 5**: Plan is tailored to specific processing type, multi-modal throughout, processing time built in at every stage
- **Score 4**: Plan reduces processing demands in most activities, mentions specific accommodations
- **Score 3**: Plan is aware of processing challenges but accommodations are vague
- **Score 1-2**: No processing accommodations, instruction is primarily verbal/fast-paced

Alert threshold: Agent score < 3 for two consecutive weeks → trigger `no_improvement` alert.
